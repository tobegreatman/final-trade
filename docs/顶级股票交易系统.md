# 顶级股票交易系统

> 理论指导实践，纪律决定成败。

---

## 一、系统总纲

### 1.1 核心哲学

| 原则 | 内涵 |
|------|------|
| **概率思维** | 不追求每笔必赢，追求正期望值的长期复利 |
| **风险管理优先** | 先求不败，后求胜。保护本金是第一要务 |
| **系统化执行** | 规则驱动，消除情绪干扰。可重复 > 可解释 |
| **适应性进化** | 市场变，系统变。持续验证，迭代优化 |
| **认知边界** | 只在能力圈内交易，不懂的不碰 |

### 1.2 系统架构

```
交易系统
├── 分析层 —— 看什么：判断市场状态与机会
│   ├── 宏观周期定位
│   ├── 市场情绪与资金面
│   ├── 行业景气度
│   ├── 个股质地筛选（含排雷）
│   └── 技术信号识别
│
├── 决策层 —— 怎么做：决定买卖什么、何时、多少
│   ├── 入场规则
│   ├── 出场规则（止盈 + 止损）
│   ├── 仓位管理（ATR动态调整）
│   └── 决策树（信号→动作流程）
│
├── 执行层 —— 动起来：纪律性下单与跟踪
│   ├── 每日执行流程
│   ├── 订单执行
│   ├── 滑点控制
│   └── 异常处理
│
└── 反馈层 —— 进化：复盘与迭代
    ├── 交易日志
    ├── 绩效归因
    ├── 回测验证
    └── 系统调优
```

## 二、市场环境判断

### 2.1 宏观周期定位（美林时钟 · 中国适配版）

> **注意：** 美林时钟基于美国市场设计，在中国需配合政策面和资金面综合判断。政策干预可能扭曲周期信号，切勿机械套用。

| 阶段 | 经济 | 通胀 | 最佳资产 | 股票策略 | 领先信号 |
|------|------|------|---------|---------|---------|
| **复苏期** | ↑ | ↓ | 股票 | 进攻：高Beta、成长股、周期股 | 社融增速连续2月回升 + PMI突破50 |
| **过热期** | ↑ | ↑ | 商品 | 收敛：资源股、通胀受益股 | M1-M2剪刀差收窄 + CPI突破3% |
| **滞胀期** | ↓ | ↑ | 现金 | 防守：低仓位、必需消费、公用事业 | PMI回落至50以下 + CPI仍高 |
| **衰退期** | ↓ | ↓ | 债券 | 等待：持有现金，准备抄底 | 政策密集出台（降准降息）= 衰退末期 |

**中国核心观察指标（按优先级排列）：**

| 优先级 | 指标 | 频率 | 判断方法 | 数据来源 |
|--------|------|------|---------|---------|
| 1 | 社融增速 | 月度 | 同比连续回升 = 宽信用开启 | 央行官网 |
| 2 | M1-M2剪刀差 | 月度 | M1增速向上穿越M2 = 资金活化 | 央行官网 |
| 3 | PMI | 月度 | >50扩张，<50收缩，关注趋势方向 | 国家统计局 |
| 4 | 政治局会议定调 | 季度 | "稳增长"= 政策底；"防风险"= 收缩预期 | 新华社 |
| 5 | 北向资金月度净流入 | 月度 | 连续净流入 = 外资看多 | 东方财富/同花顺 |
| 6 | CPI/PPI | 月度 | 剪刀差方向判断上下游利润分配 | 国家统计局 |
| 7 | 利率走向（LPR/DR007） | 月度/日 | 下行周期利好成长，上行周期利好价值 | 央行/中国货币网 |

### 2.2 市场趋势状态（多维度判定）

> **核心原则：** 不可仅凭单一指标判断趋势，至少3项信号共振才确认。

| 维度 | 牛市信号 | 震荡信号 | 熊市信号 |
|------|---------|---------|---------|
| **均线排列** | MA20 > MA60 > MA120（多头排列） | 均线缠绕，无明确方向 | MA20 < MA60 < MA120（空头排列） |
| **价格位置** | 指数收盘价 > MA60 且 MA60拐头向上 | 价格在MA60上下反复穿越 | 收盘价 < MA60 且 MA60拐头向下 |
| **创新高/新低** | 20日新高数 > 20日新低数的2倍 | 新高新低数量接近 | 20日新低数 > 20日新高数的2倍 |
| **涨跌家数** | 上涨家数/下跌家数 > 2（常态） | 涨跌各半，无持续倾向 | 下跌家数/上涨家数 > 2（常态） |
| **成交额** | 两市成交额持续放大，突破万亿 | 成交额维持在均值附近 | 成交额持续萎缩 |
| **北向资金** | 连续5日净流入 | 忽正忽负 | 连续5日净流出 |

**快速判定（做多窗口开启）：** 指数收盘站上MA60 + MA60连续3日拐头向上 + 当日上涨家数 > 下跌家数1.5倍。

### 2.3 市场情绪与资金面指标

| 维度 | 指标 | 获取方式 | 用途 |
|------|------|---------|------|
| **外资风向** | 北向资金日净流入 | 东方财富/同花顺 | 连续大额流入=看多信号 |
| **杠杆资金** | 融资余额变化（沪深两市） | 交易所官网 | 融资余额快速攀升=过热信号 |
| **游资动向** | 龙虎榜机构/游资净买入 | 东方财富龙虎榜 | 跟踪热点板块资金确认 |
| **极端情绪** | 涨停/跌停家数比 | 行情软件 | 涨停>100家=过热，跌停>50家=恐慌 |
| **情绪指数** | 恐贪指数（A股版） | 芝商所/第三方平台 | <20极度恐惧可关注，>80极度贪婪需谨慎 |

## 三、选股体系

### 3.1 四层漏斗总览

```
全市场 (5000+)
    │
    ▼ 第零层：硬性排雷（一票否决）→ 见 3.2
    │
    ▼ 第一层：基本面初筛（定量）→ 见 3.3
    │
    ▼ 第二层：景气度验证（资金+业绩）→ 见 3.4
    │
    ▼ 第三层：技术面确认（买入信号）→ 见 3.5
       │
       最终标的池 → F10深度复核 → 盈亏比计算 → 买入
```

---

### 3.2 第零层：硬性排雷（一票否决）

> 任一条件触发，直接排除。分为"选股器可筛"和"F10手动查"两部分。

**东方财富一句话选股（选股器自动筛）：**

```
非ST且非停牌且非退市且商誉占净资产比例小于30%且大股东质押比例小于60%且上市天数大于250天且最近一期审计意见为标准无保留意见
```

**F10 手动复核（候选池出来后逐一检查）：**

| 排雷项 | 排除标准 | F10路径 |
|--------|---------|---------|
| 关联交易异常 | 关联交易/营收 > 40% → 排除 | 公司资料 → 关联交易 |
| 存贷双高 | 货币资金/总资产>30% 且 有息负债/总资产>30% → 排除 | 资产负债表，手动算 |
| 董监高减持 | 近一年累计减持 > 总股本5% → 排除 | 股东研究 → 高管增减持 |
| 经营现金流异常 | 最近一年经营现金流为负 → 排除 | 现金流量表 |
| 诉讼/处罚 | 近一年重大诉讼或监管处罚 → 排除 | 公司资料 → 重大事项 |

---

### 3.3 第一层：基本面初筛（定量）

> 排雷通过后，用以下条件排除 90% 标的。

**核心指标：**

| 维度 | 指标 | 标准 | 用途 |
|------|------|------|------|
| 盈利能力 | ROE | > 12%（连续3年） | 筛出真正的赚钱公司 |
| 成长性 | 营收/利润增速 | > 15% | 确认处于增长通道 |
| 财务健康 | 资产负债率 | < 60%（非金融） | 排除高杠杆风险 |
| 现金流 | 经营现金流/净利润 | > 0.7 | 验证利润真实性 |
| 估值 | PEG | < 1.2 | 性价比 |
| 护城河 | 毛利率趋势 | 稳定或提升 | 竞争优势可持续性 |
| 管理层 | 十大股东变动 | 无异常减持 | 排雷信号 |

**东方财富一句话选股：**

```
ROE大于12%且营收同比增长大于10%且净利润同比增长大于10%且资产负债率小于60%且经营现金流为正且市盈率5到40且流通市值大于30亿且商誉占净资产比例小于30%且质押比例小于60%且非ST且非停牌
```

> 结果：约 200-400 只。

### 3.4 第二层：景气度验证（资金 + 业绩）

> 在基本面过关的标的中，找到资金认可、业绩向向上的标的。再筛 80%。

**可选方向（按需选一个叠加到第一层结果上）：**

| 方向 | 东方财富一句话选股 |
|------|------------------|
| **机构增持** | 机构持股比例较上季度增加且北向资金持股比例增加且近30日主力净流入 |
| **业绩超预期** | 最新报告期净利润同比增长大于30%且营收同比增长大于20%且近60日研报数量大于3 |
| **龙虎榜强势** | 近3日龙虎榜上榜且买方机构席位数量大于卖方机构席位且换手率大于3%小于20%且流通市值大于30亿 |

> 结果：约 30-80 只。

### 3.5 第三层：技术面确认（买入信号）

> 前两层过关后，等出现对应买入策略的技术信号再出手。
> 以下按策略分类，每条可叠加到前两层结果上。

**趋势突破信号：**

```
今天放量突破20日高点且MACD金叉且MACD柱为正且20日均线向上且60日均线向上
```

**回调买入信号：**

```
股价在60日均线上方且60日均线向上且近5日缩量回调至20日均线附近且今日成交量为近20日最低
```

**底部右侧确认信号：**

```
股价从半年高点下跌超过40%且近5日成交量萎缩后今日放量且RSI从30以下拐头向上且MACD金叉
```

**PC端自定义公式（公式管理器中粘贴）：**

趋势突破：
```
C>REF(HHV(H,20),1) AND V/REF(MA(V,20),1)>1.5
AND MACD.DIF>MACD.DEA AND MACD.MACD>0
AND MA(C,20)>MA(C,60) AND MA(C,60)>REF(MA(C,60),5);

回调到20日均线：
```
C>MA(C,60) AND MA(C,60)>REF(MA(C,60),5)
AND L<=MA(C,20)*1.02 AND C>=MA(C,20)*0.98
AND MA(V,5)<MA(V,20)*0.7;
```

回调到60日均线：
```
C>MA(C,60) AND MA(C,60)>REF(MA(C,60),5)
AND L<=MA(C,60)*1.02 AND C>=MA(C,60)*0.98
AND MA(V,5)<MA(V,20)*0.7;
```

底部放量长阳+RSI拐头：
```
HHV(H,120)/C>1.4
AND C/REF(C,1)>1.03
AND V/REF(MA(V,20),1)>2

### 3.6 估值锚定

| 方法 | 适用场景 | 公式/标准 |
|------|---------|----------|
| **PE 估值** | 稳定盈利公司 | PE < 行业中位数 × 0.8 为低估 |
| **PEG 估值** | 成长股 | PEG < 1 低估，1-1.5 合理，> 1.5 高估 |
| **PB 估值** | 银行、地产、周期股 | PB < 1 为安全边际 |
| **DCF 折现** | 现金流稳定的企业 | 内在价值 = Σ FCF / (1+r)^t |
| **相对估值** | 同行业横向比较 | 指标低于行业均值 + 有催化剂 = 机会 |

**核心原则：** 不追高估值，不在泡沫中买入好公司。估值决定买入价格，基本面决定买入方向。

## 四、买入策略

### 4.1 趋势突破买入法

**理论：** 价格突破关键阻力位，说明多方力量主导，趋势启动。

```
入场条件（全部满足）：
1. 收盘价 > HHV(HIGH, 20)（突破20日最高价）
   且当日成交量 > MA(VOL, 20) × 1.5（放量确认）
2. MACD(12,26,9) 的 DIF 线在 DEA 线上方（金叉状态）
   且 MACD柱状图 > 0（动能确认）
3. 大盘非熊市状态（按 2.2 多维度判定）
4. 基本面符合选股标准 + 未触发排雷清单
5. 盈亏比 >= 2:1（目标位与止损位的距离比）

买入仓位：计划总仓位的 50%（首批）
确认加仓：突破后回踩确认不破突破位，加仓 30%
趋势加速：放量创新高，加仓 20%
```

### 4.2 回调买入法

**理论：** 上升趋势中的回调到关键支撑位是低风险买点。

```
入场条件（全部满足）：
1. 大趋势向上：收盘价 > MA(CLOSE, 60)
   且 MA(CLOSE, 60) > REF(MA(CLOSE, 60), 5)（均线拐头向上）
2. 股价回调至 MA(CLOSE, 20) 或 MA(CLOSE, 60) 附近
   （偏差在 ±2% 以内）
3. 回调缩量：MA(VOL, 5) < MA(VOL, 20) × 0.7
4. 出现企稳K线（满足任一）：
   - 锤子线（下影线 > 实体的2倍）
   - 看涨吞没（今日阳线实体完全包住前日阴线）
   - 十字星（开盘价≈收盘价，且出现在支撑位）
5. 盈亏比 >= 2:1

买入仓位：计划总仓位的 60%
止损位：支撑位下方 2 × ATR(14)
确认加仓：放量反弹突破回调前高，加仓 40%
```

### 4.3 底部右侧确认买入法

> **重要定位：** 本策略为高难度策略，实盘资金不超过总资金 5%。
> 建议先用模拟盘练习至少 10 次信号后再实盘。
> 不做左侧抄底，只做右侧确认后的买入。

**理论：** 极度恐慌后的反转确认往往带来巨大收益，但必须等待右侧信号。

```
入场条件（至少满足5项，其中1-3为必选项）：
必选项：
1. 股价从高点跌幅 > 40%（空间够大）
2. 出现明确的底部形态（早晨之星、底部吞没、W底突破颈线）
3. 大盘同期见底企稳（至少不再创新低）

加分项（至少满足2项）：
4. 成交量出现地量后放量：MA(VOL, 5) 从低于 MA(VOL, 20) × 0.5
   回升至 MA(VOL, 20) × 1.0 以上
5. RSI(14) 从 < 30 回升至 > 35 且拐头向上
6. 出现基本面催化剂（政策利好、业绩超预期、行业拐点）
7. MACD 在零轴下方金叉且柱状图连续放大

买入仓位：分三批建仓
  第一批：计划仓位的 30%（右侧信号确认日）
  第二批：计划仓位的 30%（突破颈线/前期阻力位）
  第三批：计划仓位的 40%（回踩确认不破颈线）
止损位：买入价下方 3 × ATR(14)
```

### 4.4 三种策略适用场景

| 策略 | 适用市场 | 持有期 | 时间止损 | 风险等级 | 收益预期 |
|------|---------|--------|---------|---------|---------|
| 趋势突破 | 牛市/强趋势 | 1-4周 | 5日未突破前高→出场 | 中 | 高 |
| 回调买入 | 震荡上行 | 2-8周 | 8日未企稳（未回到MA20上方）→出场 | 低-中 | 中-高 |
| 底部右侧确认 | 熊转牛初期 | 1-6月 | 15日未确认（未突破颈线）→出场 | 高 | 极高 |

### 4.5 目标价与盈亏比计算

> **核心公式：盈亏比 = 目标收益空间 / 止损空间，要求 >= 2:1**

```
止损空间 = N × ATR(14)（从买入价到止损价的距离）
目标收益空间 = 目标价 - 买入价

目标价确定方法（按策略类型选择）：

趋势突破策略：
  目标价 = 突破位 + (突破位 - 前期回调最低点) × 1.0
  （即突破前一波段的涨幅等量投射）

回调买入策略：
  目标价 = 前高位置 或 MA(C, 20) 上方的阻力位
  （至少是止损空间的2倍以上才值得入场）

底部右侧确认策略：
  目标价 = 前期下跌起点到买入价的 38.2% 或 50% 斐波那契回撤位
  （用黄金分割位估算反弹目标）

示例：
  买入价 = 20元，ATR(14) = 0.8元，趋势突破（N=1.5）
  止损空间 = 1.5 × 0.8 = 1.2元（止损价18.8元）
  前波段低点18元，突破位20元，波段高度 = 20-18 = 2元
  目标价 = 20 + 2 = 22元
  目标收益空间 = 22 - 20 = 2元
  盈亏比 = 2.0 / 1.2 = 1.67 → 不足 2:1，需谨慎或等待更好的入场点
```

### 4.6 买入决策树

```
发现一只股票
  │
  ├─→ 触发排雷清单？ ── 是 → 直接跳过
  │                 └─ 否 ↓
  ├─→ 基本面符合标准？ ── 否 → 加入观察池，不操作
  │                   └─ 是 ↓
  ├─→ 大盘状态允许买入？ ── 否 → 加入观察池，等待
  │                      └─ 是 ↓
  ├─→ 出现买入信号？（突破/回调/反转） ── 否 → 设置提醒价格，等待
  │                                   └─ 是 ↓
  ├─→ 盈亏比 >= 2:1？ ── 否 → 跳过（目标空间不够）
  │                   └─ 是 ↓
  ├─→ 计算仓位：单只仓位 = min(总资金 × 2% / ATR止损幅度, 25%)
  │
  └─→ 执行买入 + 立即设定止损位 + 记录交易计划
```

## 五、卖出策略

### 5.1 止损规则（铁律，不可违反）

#### ATR 动态止损（核心方法）

> **核心改进：** 用 ATR 替代固定百分比止损，自动适应不同股票的波动特性。

```
ATR(14) = 14日平均真实波动幅度
买入价 = P
止损价 = P - N × ATR(14)

N 的取值：
  趋势突破策略：N = 1.5（紧止损，趋势股波动相对规律）
  回调买入策略：N = 2.0（适度止损，允许正常回调波动）
  底部右侧确认策略：N = 3.0（宽止损，底部波动剧烈）

示例：
  股票A：买入价 20元，ATR(14) = 0.8元
  趋势突破策略：止损价 = 20 - 1.5 × 0.8 = 18.8元（止损幅度 6%）
  回调买入策略：止损价 = 20 - 2.0 × 0.8 = 18.4元（止损幅度 8%）
```

#### 固定百分比止损（辅助方法）

> 仅用于 ATR 数据不可用时的替代方案。

| 策略类型 | 固定止损幅度 |
|---------|------------|
| 短线（趋势突破） | 买入价下方 5% |
| 中线（回调买入） | 买入价下方 8% |
| 长线（底部右侧确认） | 买入价下方 10% |

#### 其他止损规则

| 规则 | 细节 |
|------|------|
| **技术止损** | 跌破关键支撑位（均线/前低/趋势线），以先触发者为准 |
| **时间止损** | 买入后 N 个交易日未按预期运行，出场（具体天数见 4.4 表格） |
| **逻辑止损** | 买入逻辑被证伪（如业绩暴雷、政策转向），立即出场 |
| **总仓位止损** | 总账户回撤 > 10%，强制减仓至半仓；回撤 > 15%，空仓休息 |

#### 止损优先级（多止损触发时的处理）

```
当多个止损信号同时存在时，取最严格（最靠近当前价）的止损位执行：
  止损价 = min(ATR止损价, 技术止损价, 固定百分比止损价)
```

### 5.2 止盈规则

#### 跟踪止盈（趋势行情首选）

```
初始设定：
  跟踪止盈价 = 买入价 × (1 - 回撤比例)（买入当天）

每日收盘后更新（只上不下）：
  新基准价 = 持仓期间最高收盘价 × (1 - 回撤比例)
  跟踪止盈价 = max(昨日跟踪止盈价, 新基准价)

触发条件：收盘价 < 跟踪止盈价 → 次日开盘卖出

示例：
  买入价 = 20元，趋势突破策略（回撤比例8%）
  Day1：收盘21元 → 跟踪止盈 = 21 × 0.92 = 19.32（上移）
  Day2：收盘22元 → 跟踪止盈 = 22 × 0.92 = 20.24（继续上移）
  Day3：收盘20.5元 → 20.5 < 20.24? 否 → 不触发，跟踪止盈仍为20.24
  Day4：收盘19.8元 → 19.8 < 20.24? 是 → 次日开盘卖出

回撤比例参数：
  短线（趋势突破）：8%
  中线（回调买入）：15%
  长线（底部右侧确认）：20%
```

#### 目标止盈（波段行情）

| 方法 | 规则 | 适用场景 |
|------|------|---------|
| **目标止盈** | 到达预定目标位（基于阻力位/盈亏比计算）卖出1/2，剩余启动跟踪止盈 | 波段行情 |
| **信号止盈** | 出现明确卖出信号（满足任一）：<br>- 放量滞涨（量比>2但涨幅<1%）<br>- MACD(12,26,9) DIF下穿DEA（死叉）<br>- 顶部形态（黄昏之星、乌云盖顶、双顶） | 所有场景 |
| **分批止盈** | 第一目标位出30%，第二目标位出30%，剩余启动跟踪止盈 | 中长线 |

### 5.3 卖出优先级

```
1. 触发止损 → 立即卖出，不犹豫（当日收盘前或次日开盘）
2. 买入逻辑破坏 → 尽快卖出（当日或次日）
3. 止盈信号出现 → 执行止盈计划（可设条件单）
4. 发现更好机会 → 换股（仅当新机会盈亏比明显更优 ≥ 3:1）
```

## 六、仓位管理

### 6.1 总仓位控制

| 大盘状态 | 总仓位上限 | 判定条件 |
|---------|----------|---------|
| 牛市确认 | 80-100% | 均线多头排列 + 市场广度指标共振（见2.2） |
| 震荡偏多 | 50-70% | 站上60日均线但未确认趋势，市场广度指标分化 |
| 震荡偏空 | 20-40% | 跌破60日均线但未确认熊市，成交额萎缩 |
| 熊市 | 0-20% | 均线空头排列 + 市场广度指标全面走弱 |

### 6.2 个股仓位

| 规则 | 标准 |
|------|------|
| **单只上限** | 总资金的 25%（绝对硬性上限，任何情况不可突破） |
| **单只标准** | 总资金的 10-15%（常规配置） |
| **集中度** | 同行业不超过总仓位的 40% |
| **相关性** | 持仓中同产业链股票不超过 3 只（如：同属锂电池产业链的不超3只） |
| **持仓数量** | 5-10 只（<5 过于集中，>10 过于分散且管理不过来） |

### 6.3 风险预算与仓位计算（修正版）

> **核心原则：每笔交易的最大亏损不超过总资金的 2%。**

```
单只仓位 = min(总资金 × 2% / 止损幅度, 总资金 × 25%)

其中止损幅度 = N × ATR(14) / 买入价
（N 根据策略类型取 1.5 / 2.0 / 3.0，见 5.1）

示例1（低波动股，25%上限约束）：
  总资金 = 100万，买入价 = 20元，ATR(14) = 0.5元
  趋势突破策略（N=1.5），止损幅度 = 1.5 × 0.5 / 20 = 3.75%
  单只仓位 = min(100万 × 2% / 3.75%, 100万 × 25%)
           = min(53.3万, 25万) = 25万（触发25%上限约束）

示例2（中等波动股，风险预算约束）：
  总资金 = 100万，买入价 = 30元，ATR(14) = 1.2元
  回调买入策略（N=2.0），止损幅度 = 2.0 × 1.2 / 30 = 8%
  单只仓位 = min(100万 × 2% / 8%, 100万 × 25%)
           = min(25万, 25万) = 25万（两个约束同时生效）

示例3（高波动股，风险预算严格约束）：
  总资金 = 100万，买入价 = 50元，ATR(14) = 3.0元
  底部右侧确认策略（N=3.0），止损幅度 = 3.0 × 3.0 / 50 = 18%
  单只仓位 = min(100万 × 2% / 18%, 100万 × 25%)
           = min(11.1万, 25万) = 11.1万（风险预算严格约束）
  → 高波动股自然限制仓位，无需人为干预
```

### 6.4 加减仓规则

| 操作 | 条件 | 动作 |
|------|------|------|
| **加仓** | 浮盈 > 1.5 × ATR(14)（相对买入价）+ 趋势延续确认 | 金字塔加仓（每次递减：如首批40%，加仓30%，再加20%） |
| **减仓** | 浮盈从最高点回撤超过50%（如最高浮盈10%，回撤至5%以内）或信号走弱 | 减至半仓 |
| **清仓** | 触发止损或逻辑破坏 | 全部卖出 |
| **补仓禁止** | 亏损时绝不在亏损方向加仓 | 止损而非补仓 |

## 七、交易纪律

### 7.1 交易前检查清单

```
□ 大盘环境是否适合交易？（对照 2.2 多维度判定表）
□ 该股是否通过排雷清单？（对照 3.2 排雷）
□ 该股基本面是否符合标准？（对照 3.2 指标表）
□ 是否有明确的买入信号？（对照 4.1/4.2/4.3 入场条件）
□ 止损位是否已设定？（用ATR计算，并设条件单）
□ 盈亏比是否 >= 2:1？
□ 仓位是否在风险预算内？（用 6.3 公式计算，不超过25%）
□ 是否在盘中冷静状态下决策？（不在开盘/收盘前10分钟冲动决策）
□ 是否已设条件单/提醒？（不依赖盘中临时判断）
```

### 7.2 铁律（零容忍）

| 编号 | 规则 | 违反后果 |
|------|------|---------|
| 1 | **止损不可撤销** — 到了止损位必须卖，不抱任何幻想 | 小亏变大亏，最终大亏 |
| 2 | **不追涨杀跌** — 没有信号不进场，不因 FOMO 买入 | 高位接盘/低位割肉 |
| 3 | **不逆势满仓** — 熊市不加杠杆，震荡市不满仓 | 爆仓/深度套牢风险 |
| 4 | **不在亏损仓位加仓** — 亏了不加码，止损而非补仓 | 亏损放大，复利杀手 |
| 5 | **不过度交易** — 有信号才交易，每月交易不超过 10 次 | 手续费+滑点吞噬利润 |
| 6 | **不碰不懂的票** — 没研究过的不买，不跟风听消息 | 盲目赌博，必亏 |
| 7 | **不临时起意** — 盘中不产生新想法，所有计划盘前制定 | 情绪化操作，破坏系统 |

### 7.3 心理管理

| 情绪 | 触发场景 | 应对方法 |
|------|---------|---------|
| **贪婪** | 连续盈利，想加仓加杠杆 | 回顾规则，严格执行仓位上限 |
| **恐惧** | 连续亏损，不敢下单 | 缩小仓位恢复信心，检查系统是否失效 |
| **后悔** | 卖出后继续涨 / 未买入的涨了 | 接受不确定性，关注过程而非结果 |
| **侥幸** | 亏损持仓期待回本 | 执行止损，市场不关心你的成本 |
| **疲劳** | 盯盘过度，决策质量下降 | 设定交易时段，其余时间不盯盘 |

## 九、绩效评估与系统迭代

### 9.1 核心绩效指标

| 指标 | 公式 | 合格标准 | 优秀标准 |
|------|------|---------|---------|
| **超额收益** | 组合收益 - 沪深300收益 | > 0%（跑赢基准） | 超基准 5%+ |
| **年化收益率** | 几何平均 | > 12% | > 20% |
| **最大回撤** | 峰值到谷底最大跌幅 | < 15% | < 10% |
| **夏普比率** | (收益率-无风险)/波动率 | > 0.8 | > 1.5 |
| **卡尔玛比率** | 年化收益/最大回撤 | > 1.0 | > 2.0 |
| **胜率** | 盈利次数/总次数 | > 45% | > 55% |
| **盈亏比** | 平均盈利/平均亏损 | > 1.5:1 | > 2.5:1 |
| **月度胜率** | 盈利月数/总月数 | > 60% | > 70% |

> **基准说明：** 个人交易者的合理目标是**持续跑赢沪深300指数 5-10个百分点**。
> 不建议以绝对收益率（如"年化25%"）为目标，这会导致在牛市低估自己、在熊市冒险追求。

### 9.2 系统失效判定

```
以下任一情况出现，暂停交易，全面复盘：

1. 连续 5 次交易止损 → 检查选股/买入条件是否失效
2. 单月亏损 > 8% → 检查仓位管理是否合规
3. 最大回撤 > 15% → 降低仓位，回归模拟盘
4. 月度胜率连续3月 < 30% → 系统可能不适应当前市场
5. 绩效连续3月跑输沪深300 → 重新评估策略有效性
```

### 9.3 回测与验证框架

> **核心原则：** 任何策略调整必须经过回测验证，不凭感觉修改系统。

#### 回测要求

| 项目 | 最低要求 |
|------|---------|
| 回测时间跨度 | 至少 3 年（覆盖牛熊周期） |
| 样本量 | 至少 50 次交易信号 |
| 滑点假设 | 每笔交易扣除 0.5% 成本（佣金约0.05%×2 + 印花税0.05%卖出 + 滑点约0.1%×2） |
| 资金曲线 | 记录每笔交易后的资金曲线，计算最大回撤 |
| 信号 vs 执行分离 | 区分"信号胜率"和"执行胜率"，找出差距 |

#### 推荐工具

| 工具 | 适用场景 | 入门门槛 |
|------|---------|---------|
| 通达信选股公式 | 快速回测选股条件 | 低 |
| Python + Tushare/AKShare | 自定义策略回测 | 中 |
| 聚宽/米筐 | 在线量化平台，可视化回测 | 中 |
| Excel | 简单的手动回测 | 低 |

### 9.4 迭代流程

```
月度复盘 → 识别亏损交易的共性原因
    │
    ├── 执行问题（未守纪律）→ 强化规则，不改动系统
    │   标志：信号正确但执行偏差 > 2%
    │
    ├── 选股问题（标的质量差）→ 调整筛选条件
    │   标志：胜率低但盈亏比正常
    │
    ├── 时机问题（买卖点不准）→ 优化技术信号参数
    │   标志：胜率低且盈亏比低
    │
    └── 市场问题（系统不适应）→ 暂停该策略，切换市场状态应对方案
        标志：连续3月跑输基准
```
